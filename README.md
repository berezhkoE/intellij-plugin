# Internship-Test

<!-- Plugin description -->
Following actions were added into EditorPopupMenu group:
* Show Current Project and File Names
* Show Current Method Info (available only for Kotlin files)

Notification about file extension implemented as Editor Banner (editorNotificationProvider)
<!-- Plugin description end -->

## Installation

- Using IDE built-in plugin system:
  
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "intellij-plugin"</kbd> >
  <kbd>Install Plugin</kbd>
  
- Manually:

  Download the [latest release](https://github.com/berezhkoE/intellij-plugin/releases/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
